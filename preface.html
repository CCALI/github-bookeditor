Background Information

Github is a repository for sharing code.

Connexions is a repository for sharing free textbooks in many languages.

Philip writes code for Connexions.

Motivation

Github does certain things well

This project lets people write their books in Github

Benefits of Each

Github makes it easy to do certain things:

  • Derive a project (fork)
  • Submit changes from derived projects
  • Comment on changes before accepting them

Connexions makes it easy to do certain things:

  • Remix parts of different books together into a new book
  • Derive parts of a book
  • Create various output formats
  • Manage attribution to properly attribute derived and remixed books

Drawbacks of Each

Github is aimed at programmers:

  • Difficult to understand a diff of HTML
  • Tech specific concepts (fork, pull, commit, patch, branch, merge, directory tree)

Connexions:

  • Human resources
  • Legacy content (archives are a pain)
  • Slow
